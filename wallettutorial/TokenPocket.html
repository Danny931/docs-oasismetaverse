TokenPocket

 

1. Download TP wallet

Visit TokenPocket wallet official website to download https://www.tokenpocket.pro.

Install and then open TP wallet.

2. Connect Wallet

If you already have a TP wallet, you can choose to import wallet, select Binance Smart Chain.

You could also choose private key import，mnemonic words or keystore three ways.

​

If you don’t have a TP wallet, you could choose to create one, select Binance Smart Chain.

Set up the name of your wallet, code etc., and please be sure to keep your mnemonic words safe.

Go to the main wallet page and switch to Binance Smart Chain wallet.

3. Use KSwap

Open your TP wallet, click Discover page. Click Search box.

Input KSwap website: https://app.oasiswap.finance and go to it, then you can use KSwap.
